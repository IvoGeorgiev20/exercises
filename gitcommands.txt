git clone - използва се за получаване на хранилище от дадения URL адрес
git add - команда за добавяне на файлове 
git commit - записва файлът/файловете перманентно
git push - изпраща запазените промени в хранилището (repository-то)
git init - създава локално repository(хранилище)
git status - изписва променените файлове и тези, които трябва да се add-нат или comitt-нат
git remote -v - дава списък на всички "отдалечени" хранилища(тези, които са записани на компютъра) 
git remote add origin <URL> - добавя се URl-а на repository-то, за да може промените да се добавят там
git pull - изтегля промените от github и ги качва на папката, която е свързана с repository-то
git diff - показва всички направени промени във локалното repository
git log - показва всички направени промени във github и кой ги е направил
